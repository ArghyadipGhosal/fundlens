Data Sources & Methodology
Live price data sourced from Yahoo Finance public API via open CORS proxies. Where live data is unavailable for a specific ticker (e.g. non-US mutual funds), a representative ETF proxy is used and labelled accordingly. Risk metrics use trailing daily log returns. Sharpe Ratio: risk-free rate 4.5% p.a. (approx. 3M T-Bill). Max drawdown: peak-to-trough. Beta and Alpha: CAPM, daily returns over the period.
